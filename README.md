# Space-War-Unity
A space shooter made in Unity 2019. 

## Controls
* Arrow Keys: Maneuver
* Space: Fire bullets
* B: Launch Bomb
* L: Fire Laser
* J: E-brake turn

* 0-6: Spawn agents on respective teams (hold shift to remove agents)
* A: Spawn asteroid
* T: Teleport to mouse location
* S: Summon (script on gameObject w/ rigidbody)
* Plus/Minus: Change camera zoom
